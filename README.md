### ðŸ”¨ Installation Guide

Follow these steps to install and use the application.

**Requirements**

Software:

- [Node.js](https://nodejs.org/en/download) (version 20.10.0 or higher)

Hardware:

- RAM: 4GB or higher
- CPU: Any modern processor

**Preparation**

- Clone this repository and install dependencies:
	```bash
	git clone https://github.com/hdkhanh462/turbo-p2p-share
	cd turbo-p2p-share
	npm install
	```
- Update environment variables in `.env` (see `.env.example` if available)
- Start the development server for both web and server:
	```bash
	npm run dev
	```

**Running the App**

- Open [http://localhost:3001](http://localhost:3001) in your browser for the web interface.
- The server will run on [http://localhost:3000](http://localhost:3000) by default.
